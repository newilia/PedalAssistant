# PedalAssistant

Программа для мониторинга осей игровых устройств (джойстиков, педалей, рулей) с звуковым оповещением при попадании значения в заданный диапазон.

## Функции

- Отображение списка подключенных игровых устройств
- Визуализация значений всех осей выбранного устройства
- Множество обработчиков на каждую ось с индивидуальными настройками
- Настройка минимального и максимального порога для каждого обработчика
- Звуковое оповещение при попадании значения в диапазон
- Настройка тона: частота, громкость, форма сигнала (синус, пила, меандр)
- Одновременное воспроизведение нескольких тонов (микширование)

## Установка

1. Убедитесь, что установлен Python 3.8+
2. Запустите `run.cmd` — зависимости установятся автоматически

## Запуск

Дважды кликните на `run.cmd` или выполните:

```bash
python pedal_assistant.py
```

## Использование

1. Выберите игровое устройство из выпадающего списка
2. Появятся шкалы для всех осей устройства
3. Нажмите "Добавить обработчик" под нужной осью
4. Настройте диапазон срабатывания (Мин/Макс) и параметры звука
5. При попадании значения оси в диапазон прозвучит настроенный тон
